# kebab-tracker-bot

To install dependencies:

```bash
bun install
```

Create your local environment file:

```bash
cp .env.example .env
```

To run:

```bash
bun run dev
```

Type-check (no emit):

```bash
bun run typecheck
```

This project was created using `bun init` in bun v1.3.1. [Bun](https://bun.com) is a fast all-in-one JavaScript runtime.
